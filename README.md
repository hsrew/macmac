# 🎬 영상 다운로더 & 갤러리 (macOS 스타일)

유튜브와 인스타그램 영상을 쉽게 다운로드하고 갤러리처럼 관리할 수 있는 웹 애플리케이션입니다.
**PyQt5 GUI 컨트롤러 + macOS 스타일 웹 인터페이스 + 보안 로그인**

## ✨ 주요 기능

### 🔐 **보안 기능**
- 🚀 **위장 로그인 페이지**: 도형들이 떠다니는 페이지, 로켓(🚀)을 3번 클릭하면 로그인 페이지 나타남
- 🔒 **로그인 인증**: ID/PW 없이는 접속 불가
- 🚪 **로그아웃**: 안전한 세션 관리

### 🖥️ **GUI 서버 컨트롤러**
- 🎮 포트 직접 입력 (기본: 7777)
- 🚀 서버 시작/중지
- 🌐 사이트 바로 접속
- 📁 영상 폴더 열기
- 📊 실시간 서버 로그

### 🎬 **영상 관리**
- 📥 유튜브 + 인스타그램 다운로드 (YouTube Shorts 지원)
- 📸 자동 썸네일 생성 (1초 지점)
- 📱 **3줄 갤러리** (크고 보기 편함)
- 🔍 검색 기능 (제목, 플랫폼)
- 🗑️ 삭제 기능 (hover 시 우측 상단)

### 🍎 **macOS 스타일 UI**
- 글래스모피즘 (반투명 + 블러)
- 세련된 그라데이션
- 부드러운 애니메이션
- SF Pro Display 폰트 스타일
- 정교한 그림자 효과

## 🔐 로그인 방법

### 1️⃣ **위장 페이지**
- 처음 접속 시 도형들이 떠다니는 페이지 표시
- 아무것도 아닌 것처럼 보임

### 2️⃣ **로켓 찾기**
- 화면에 떠다니는 **로켓(🚀)** 찾기
- 다른 도형들: 🔵 🔺 ⬛ ⭐ 💠 🔶 ⬜

### 3️⃣ **3번 클릭**
- 로켓을 **3번 연속 클릭**
- 클릭할 때마다 약간 커짐
- 3초 안에 3번 클릭해야 함

### 4️⃣ **로그인**
- 진짜 로그인 페이지 나타남
- 아이디와 비밀번호 입력
- "🚀 로그인" 버튼 클릭

### 5️⃣ **접속 완료**
- 메인 페이지로 이동
- 영상 다운로드 시작!

## 🔑 로그인 정보

## 📱 사용 방법

### 접속 주소

#### 맥북:
```
http://localhost:7777
```

#### 핸드폰 (같은 WiFi):
```
http://192.168.0.54:7777
```

### 영상 다운로드
1. 로그인 후 메인 페이지
2. URL 입력창에 유튜브/인스타그램 URL 붙여넣기
3. "다운로드" 버튼 클릭
4. 완료 후 갤러리에 자동 표시

### 검색
- 갤러리 상단 검색창에 검색어 입력
- 제목이나 플랫폼(youtube, instagram)으로 검색

### 삭제
- 영상에 마우스 올리면 우측 상단에 🗑️ 버튼
- 클릭 후 확인 → 삭제

## 🎨 macOS 스타일 특징

### 디자인 원칙
- ✅ 글래스모피즘 (반투명 + 블러)
- ✅ 큰 border-radius (12px-24px)
- ✅ 부드러운 transition (cubic-bezier)
- ✅ 다층 그림자로 깊이감
- ✅ SF Pro 폰트 스타일

### 갤러리 레이아웃
- **PC**: 3줄 배치, 넓은 간격 (16px)
- **모바일**: 3줄 배치, 좁은 간격 (8px)
- **카드 크기**: 충분히 크게 (제목 읽기 쉬움)

## 🛠️ 기술 스택

- **GUI**: PyQt5
- **백엔드**: Flask + Python 3.9+
- **영상**: yt-dlp (최신), instaloader (4.14.2)
- **이미지**: OpenCV, Pillow
- **프론트**: HTML5, CSS3, JavaScript
- **보안**: Flask Session
